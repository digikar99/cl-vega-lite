#+title: Component Hierarchy

Let us see the component and slot hierarchy of vega-lite using the closer-mop. First, we load =vega-lite-wip=.

#+begin_src lisp
(ql:quickload "vega-lite-wip")
#+end_src

Next, let us define a package for us to play around in:

#+begin_src lisp
(defpackage :vega-introspection
  (:use :cl))
#+end_src

Now, let us use =closer-mop= to traverse the class and slot hierarchy, starting from the =vega-lite:plot= class. We will limit to =*max-depth*= to keep things manageable.

#+begin_src lisp
(in-package :vega-introspection)

(defparameter *max-depth* 2)

(defun traverse-slots (class &optional (depth 0))
  (closer-mop:ensure-finalized class)
  (when (<= depth *max-depth*)
    (format t "~A" (string-downcase (class-name class)))
    (terpri)
    (let ((slots (closer-mop:class-slots class)))
      (pprint-logical-block (*standard-output* nil :per-line-prefix "    ")
        (loop :for slot :in slots
              :for name := (closer-mop:slot-definition-name slot)
              :for class := (find-class name nil)
              :do (if class
                      (traverse-slots class (1+ depth))
                      (progn
                        (format t "~A" (string-downcase name))
                        (terpri)))))
      (terpri))))

(traverse-slots (find-class 'vega-lite:plot))
#+end_src
